# hello-docker-compose

### Precondition

- Docker

### Architecture

![docker-compose.jpg](https://i.loli.net/2020/05/06/8WmJuRzsC32vTMD.jpg)

### Run with docker

```sh
$ docker-compose up -d
```

### Stop run

```sh
$ docker-compose down
```

### Stop run and remove data

```sh
$ docker-compose down --volume
```
